# RAG Learning

Step-by-step experiments learning Retrieval-Augmented Generation.
Concepts learned here will be implemented in AI-Chatbot repo.

## Progression
- 01-embeddings    — text to vectors, cosine similarity, Chroma basics
- 02-vector-db     — Chroma advanced, metadata filtering, persistent storage
- 03-rag-pipeline  — retrieve + generate, basic Q&A over documents
- 04-pdf-rag       — PDF loading, chunking strategies, PDF Q&A
- 05-advanced-rag  — re-ranking, hybrid search, conversation memory
